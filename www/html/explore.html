Time Range

Enter an RFC3339 style start or end timestamp. See the Go time package for more details.

Aggregation

Select the level of aggregation to apply to each series of values in the database. If `none` is chosen each series of values will be represented individually. The value should be a parsable duration string. See the Go ParseDuration documentation for more details.

1s 5m 24h 48h

Function

Apply a specific function against each series of values before plotting them.
